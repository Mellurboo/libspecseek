# libspecseek by Mellurboo
a library dedicated to accessing and referencing hardware specifications in a neat C static library.

with this library you can access various features of the hardware your program is running on from CPU Cache information to Specific features that are supported by the processor.

> [!NOTE]  
> this is simply a library, not an executable. if you are looking to view your hardware specifications, please check out [Mellurboo/specseek](https://github.com/Mellurboo/SpecSeek) or feel free to write your own!
